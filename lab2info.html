Lab Requirements

This Lab requires the implementation of the following elements:

  • Setup of <DOCTYPE>, <html>, <head>, and <body>
  • Configuration of the title
  • Use of structural tags to lay out pages - <header>, <nav>, <main>, and <footer>
  • Use of <div> to subdivide main
  • Under <nav>:
    • use anchor tags to create common site navigation (use relative hyperlinks)
  • Under <header>:
    • use h1 tag
  • Under <footer>:
    • Use of <h3> or <h4> tags
    • Use of special characters
    • An email link
  • Under <main>:
    • in 1 file only: create a <ul>
    • in 1 file only: create an <ol>
    • in 1 file only: create a <dl>
    • add absolute hyperlink to external site of your choice
